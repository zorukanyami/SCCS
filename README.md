# SCCS (Shuffle Code Claimer Service)

SCCS is an automated promo code capturing and claiming tool developed for the Shuffle.com platform.

## Features
- **Automated Capturing:** Instantly detects codes shared in Telegram channels.
- **Fast Claiming:** Claims codes to your account within seconds.
- **Secure:** Keeps your account safe thanks to browser-based operation.
- **Licensed Usage:** HWID-based secure licensing system.

## Installation
1. **Download:** Download the latest version (`.exe`) from the Releases page.
2. **Install:** Run the downloaded `SCCS setup.exe` file and follow the steps.
3. **Launch:** Start the application from the desktop shortcut.

## Usage
1. Copy your **HWID** number that appears on the Installation screen or License window.
2. Start the Bot by clicking the "Buy License" button in the app or visiting [SCCS Bot](https://t.me/ShuffleCodeClaimerService_Bot).
3. Send your copied HWID number to the Bot and follow the instructions.
4. Once your license is activated, click the "Check License" button in the app.
5. Log in to your Shuffle.com account in the window that opens.
6. The program will now run in the background and collect codes for you.
